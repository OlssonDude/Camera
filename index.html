Instruktioner

Manual, kälkod och körbara jar filer kan laddas ner genom att följa länkarna ovan.

Källkoden kan importeras till eclipse genom att välja

  1. File
  2. Import
  3. Existing project into workspace
  4. Select archive file, och välj zip filen med källkod.

För att kunna använda de körbara jar-filerna måste man följa instruktionerna här för att instalera och starta proxyserver. Sedan kör man följande kommandon i terminalen där jar-filerna finns för att starta serven java -jar server.jar host port Där host är numret på kameran man valde, alltså N i rt@argus-N, och port är porten man valde. Därefter java -jar client.jar för att starta klienten.
